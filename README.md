## Ticketing diary
### 프로젝트 제작 이유
기존의 예매 사이트들은 예매 기능만 있는 사이트가 대부분이었습니다. 그런 사이트가 아닌 공연에 대한 느낀 점 및 자신만의 기록을 남길 수 있는 서비스를 제공하면 사용자들이 공연에 대한 기억을 오래 간직할 수 있을 것이라는 생각으로 제작하게 되었습니다.
